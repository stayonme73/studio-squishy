# Welcome Hall Pivot — Three-Screen Journey

**Status:** Founder approved  
**Date:** June 2026

## Principle

Each screen has **one clear job**. The customer never feels like they are working.

| Screen | Job |
| ------ | --- |
| **Welcome Hall** | Wonder — welcome, impress, reassure, curiosity |
| **Studio Guide** | Understanding — compare packages, choose |
| **Draft Room** | Action — confirm, intake, begin |

## Phase 1: Welcome Hall (`/`)

- Approved atmosphere (black hall, tower, left wall, city, industrial ceiling).
- **Not** responsible for package education or decision wall.
- **Tower** is the primary transition into Studio Guide; kiosk is secondary.
- Transition: tower glow → forward into View Ahead → Studio Guide emerges.
- Flow: Enter → experience → tap tower or kiosk → Studio Guide.

## Phase 2: Studio Guide (`/studio-guide`)

- Separate focused experience (usability over environmental storytelling).
- Tower: Spark · Momentum · Growth.
- Readable wall: large text, laptop-friendly, scannable.
- **HOW CAN WE HELP?** → Draft Room with package context.

## Phase 3: Draft Room (`/draft-room`)

- Intake, confirmation, pricing review, campaign initiation.

## Preserved work

V3 showroom components (`ShowroomTower`, `ShowroomWall`) move to **Studio Guide** — not discarded.
